Structure itérative
===================

Une **structure itérative** est un code qui permet de répéter des instructions un grand nombre de fois en les écrivant qu'une seule fois. Une structure itérative est aussi appelée **boucle**.

En python, il existe 2 boucles différentes:

-  La boucle ``for`` qui est boucle bornée;
-  La boucle ``while`` qui est une boucle conditionnelle.
